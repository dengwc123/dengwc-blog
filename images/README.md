# 网站所需图片

请准备以下图片并放置在此文件夹中：

1. `profile.jpg` - 个人照片
   - 建议尺寸：500x500像素
   - 最好是正方形图片
   - 清晰的头像照片

2. `slide1.jpg`, `slide2.jpg`, `slide3.jpg` - 轮播图照片
   - 建议尺寸：1920x1080像素
   - 横向照片
   - 高质量、主题相关的图片

3. `project1.jpg`, `project2.jpg`, `project3.jpg` - 项目展示图片
   - 建议尺寸：800x600像素
   - 项目截图或相关图片
   - 清晰的展示图

注意事项：
- 所有图片都应该是高质量的
- 确保您拥有这些图片的使用权
- 图片大小建议控制在500KB以内
- 使用 JPG 格式以获得更好的压缩率
